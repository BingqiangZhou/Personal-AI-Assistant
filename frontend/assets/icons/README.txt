How to use (Xcode):
1) In Xcode, open Assets.xcassets.
2) Drag the whole 'AppIcon.appiconset' folder into Assets.xcassets (or replace your existing AppIcon set).
3) Ensure your target's App Icons Source is set to that AppIcon asset.

Included:
- AppIcon.appiconset (all required iPhone + iPad + App Store sizes)
- MindriverIcon-1024.png (master)
